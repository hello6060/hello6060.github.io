用Python写一个简单的交互式SSH客户端

用Python写一个简单的交互式SSH客户端

在学习了python自动脚本后, 就想写一个简单的shell

使用的是paramiko包

简介

ssh是一个协议，OpenSSH是其中一个开源实现，paramiko是Python的一个库，实现了SSHv2协议(底层使用cryptography)。

有了Paramiko以后，我们就可以在Python代码中直接使用SSH协议对远程服务器执行操作，而不是通过ssh命令对远程服务器进行操作。

paramiko包含两个核心组件：SSHClient和SFTPClient。

在自动脚本中命令间是隔离的, 但我想要程序能做到交互的功能, 所以要保持住ssh的状态, 之前的exec_command函数无法使用了

这里我使用了invoke_shell函数(区别在于invoke_shell使用SSH shell通道,而exec_command使用SSH exec通道)

代码

import paramiko, time

def send(cmd, channel):
    cmd += '\n'
    result = ''
    channel.send(cmd)

    # 有些命令执行需要较长时间, 这里使用循环做回显等待
    while True:
        time.sleep(0.2)
        # recv会将我们输入的cmd带回来, 将其切割掉
        res = channel.recv(65535)[len(cmd):]
        res = res.decode('utf-8')
        result += res
        print(res,end="")
    # 判断通道是否已经空闲
        if not channel.recv_ready():
            break
    return result


if __name__ == '__main__':
    ssh = paramiko.SSHClient()
    ssh.set_missing_host_key_policy(paramiko.AutoAddPolicy())  # 忽略主机key
    ssh.connect('10.10.10.200', port=22, username='root', password='000000', compress=True)

    channel = ssh.invoke_shell()
    send("\n", channel)

    while True:
        cmd = input("")
        send(cmd,channel)
